## 회신 핵심 내용 요약

### 회신 1: 양식 자동화의 의미 재정의 (★ 가장 중요)

강사는 설문 결과 *"업무 문서 자동 작성 75%"*를 **출장보고서·기획안 등 정형 양식 자동 작성**으로 해석했음.

회신 결과 — 실제 의미는 다름:

> *"해당 니즈는 정해진 사내 양식을 자동 작성하는 의미라기보다는 생성형 AI를 활용한 텍스트·콘텐츠 작성 업무에 가까운 것으로 파악되었습니다."*

→ **강의 spine 일부 수정 필요**:
- 기존: *"내 양식 통합"*
- 변경 후: *"내 콘텐츠 통합"* (양식 → 콘텐츠 생산으로)

### 회신 2: 실제 임직원 업무 영역 — 두 목록 비교

회신에는 업무 목록이 **두 번** 나옵니다. 둘이 살짝 다르므로 정확히 짚고 갑니다.

#### [목록 ①] 현재 임직원이 실제로 하는 업무 예시 (7개)

> 발신자가 "예를 들면 다음과 같은 업무입니다"라며 제시

1. 신문광고용 여행상품 슬로건 작성
2. **슬로건 캘리그라피 이미지 제작**
3. 여행상품 핵심 포인트 및 홍보 문구 작성
4. 기사광고 문안 작성
5. 그룹웨어 기안서 본문 초안 작성
6. 기획전 문구 및 구성안 작성
7. 엑셀·PPT 자료 작성 보조

#### [목록 ②] 강의 실습용으로 추천된 업무 영역 (7개) — ★ 강의 설계 기준

> 발신자가 "실무형 프롬프트 실습 중심으로 구성해주시면 학습 효과가 더 높을 것"이라며 제시

1. **여행상품 홍보 문구 작성**
2. **상품별 핵심 세일즈 포인트 정리**
3. **기사광고·신문광고 문안 작성**
4. **그룹웨어 기안문 초안 작성**
5. **기획전 콘셉트 및 카피 작성**
6. **엑셀 데이터 요약 및 보고용 문장 작성**
7. **PPT 목차·슬라이드 문안 구성**

#### 두 목록 차이 분석

| 변경 유형 | 목록 ① | 목록 ② | 의미 |
|---|---|---|---|
| **빠진 항목** | 슬로건 캘리그라피 이미지 제작 | (없음) | 이미지 생성은 추천 실습에서 제외 |
| **정제·분리** | 여행상품 핵심 포인트 및 홍보 문구 작성 | ① 홍보 문구 + ② 세일즈 포인트 정리 (둘로 분리) | 두 업무로 명확화 |
| **통합** | 신문광고용 슬로건 + 기사광고 문안 | ③ 기사광고·신문광고 문안 작성 | 광고류 통합 |
| **정제** | 기획전 문구 및 구성안 | ⑤ 기획전 콘셉트 및 카피 작성 | "문구→카피", "구성안→콘셉트"로 정제 |
| **분리·세분화** | 엑셀·PPT 자료 작성 보조 | ⑥ 엑셀 데이터 요약 및 보고용 문장 + ⑦ PPT 목차·슬라이드 문안 (둘로 분리) | 분석·문안으로 세분 |
| **표현 변경** | 그룹웨어 기안서 본문 초안 | ④ 그룹웨어 기안문 초안 작성 | "기안서→기안문" |

→ **목록 ②가 발신자의 정제된 최종 권고안**. 강의 설계는 목록 ②를 기준으로 함.

#### 캘리그라피 처리 방침

- 목록 ①에만 있고 목록 ②에서 빠짐 — **메인 실습 영역 아님**
- 단, 이미지 생성 AI 수요 자체는 회신에서 인정됨
- **처리**: 2차시 E 카피 마스터 실습의 **숙련자 보너스**로 GPT 호명 ("캘리그라피·슬로건 이미지는 GPT가 강합니다" 30초 안내)

### 회신 3: 가정산 업무는 EIP 시스템 처리

> *"행사 종료 후 가정산 업무는 EIP 프로그램에 입력·처리하는 방식으로 운영"*

→ AI 활용 영역 아님. 강의 범위 외.

### 회신 4: 엑셀·PPT 양식은 팀별 제각각

> *"실제 사용하는 양식은 팀별·업무별로 제각각이어서 회사 공통 서식으로 제공할 만한 자료를 특정하기 어렵습니다."*

→ 표준 양식 제공 불가. 4차시 데이터 분석은 더미 데이터로만.

### 회신 5: 양식 제공은 불가, 더미 시나리오 협의는 가능

> *"실제 사내 문서 양식이 아닌, 위 업무 유형을 반영한 더미 사례나 실습 주제 수준으로는 추가 협의가 가능할 것 같습니다."*

→ **강사가 더미 시나리오 작성 → 검수 요청 → 코멘트 받기 가능**.
→ 강의 자료에 *"롯데관광 사전 검수 완료"* 라벨 가능.

## 강의 설계 변경 사항

### 1. Narrative spine 수정

| 변경 전 | 변경 후 |
|---|---|
| 내 양식·내 문서·외부 정보·내 지식 | **내 콘텐츠·내 문서·외부 정보·내 지식** |

### 2. 1차시 D 블록 시연 시나리오 교체

| 변경 전 | 변경 후 |
|---|---|
| 출장보고서 메모 + 양식 → 보고서 | **신문광고 문안 (Tona 기반)** |

기존 *"양식 Few-shot"* 시연은 **콘텐츠 생성 시연**으로 교체. Tona 프롬프트 활용.

### 3. 7개 직군별 실습 콘텐츠 재구성

| 직군 | 1차시 실습 콘텐츠 (양식 기반 X / 콘텐츠 생성 O) |
|---|---|
| 상품기획·오퍼레이터 | 신상품 홍보 카피 (광고 문안) |
| 영업·판매 | 거래선 제안 메시지 |
| 예약·발권 | 고객 변경 안내문 |
| 경영지원·인사총무 | 그룹웨어 기안문 |
| 고객서비스·VOC | 응대 답변문 |
| 카지노·VIP | VIP 안내문 |
| IT·시스템 | 시스템 변경 공지 |

## 더미 시나리오 작성 가이드 (검수 요청용)

### 작성 원칙

1. **실제 업무 흐름을 반영**하되 가상의 상품·일정·금액 사용
2. **롯데관광 톤·매너 검수 가능한 수준의 디테일** 포함
3. **각 직군 1건씩 — 강의 시연·실습 자료로 직결**

### 우선 작성할 더미 시나리오 4건 (검수 요청 패키지)

#### 1. 신문광고 더미 시나리오
- **상품**: 오사카·교토 단풍 5일 (가상 상품)
- **출발 시기**: 11월
- **가격**: 169~189만원
- **핵심 셀링**: 사찰 단풍 명소 3곳 + 료칸 1박
- **타겟**: 40~60대 부부
- **매체**: 조선일보 5단 광고
- **결과물**: 메인 카피 1줄 + 서브 1줄 + 본문 80자, 3안

#### 2. 기획전 콘셉트 더미 시나리오
- **시즌**: 가을 동남아 가족 기획전
- **대상국**: 다낭, 푸꾸옥, 코타키나발루 (가상 패키지)
- **기획전 기간**: 9~11월
- **타겟**: 초등학생 자녀를 둔 30~40대 부부
- **차별 포인트**: 키즈 클럽, 가족 룸 카테고리
- **결과물**: 기획전 콘셉트명 + 메인 카피 + 코너별 카피

#### 3. 그룹웨어 기안문 더미 시나리오
- **기안 종류**: 부서 워크숍 기안
- **부서**: 영업본부 (가상)
- **워크숍 일정**: 7월 OO일, 1박 2일
- **장소**: 경기 양평 OO리조트 (가상)
- **참석**: 약 30명
- **예산**: 약 OOO만원
- **결과물**: 그룹웨어 기안문 본문 초안

#### 4. 엑셀·PPT 문안 더미 시나리오
- **데이터**: 가상 월간 매출 데이터 (지역별·상품별)
- **요청**: 데이터에서 인사이트 3가지 도출 + 보고용 문장 + PPT 슬라이드 목차
- **타겟**: 임원 보고용
- **결과물**: 인사이트 3줄 + 보고 문장 5줄 + PPT 슬라이드 5장 목차

## 회신 메일 답변 (이미 발송)

발송한 답신의 핵심:

1. 회신 내용 감사 표명 (양식 자동화가 아닌 콘텐츠 생산이라는 정확한 정보)
2. 강의 시연·실습 시나리오를 회신의 7가지 업무로 재구성하겠다고 명시
3. 더미 시나리오 4건 작성해 검수 요청 예정 (D-OO일까지)
4. 슬로건 캘리그라피 이미지는 숙련자 보너스 코너에서 처리 검토

→ **현재 더미 시나리오 4건 작성이 후속 작업의 최우선 순위**
