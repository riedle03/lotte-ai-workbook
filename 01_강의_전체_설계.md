### A 오프닝 핵심 자료
- 하나투어 H-AI: 챗봇 이용자 **1,154% 증가**, 직접 입력 60%, 야간 사용 42%
- 하나투어 ChatGPT Enterprise: 항공권 환불 문의 **40% 감소**
- 4차시 전체 지도 1슬라이드 + 1차시 위치 표시

### B 트렌드 강의 (3슬라이드)

**B-1. AI 사용의 3단계 진화 (3분)**
- 1단계 채팅 (2022~2023): "질문 한 번 → 답 한 번"
- 2단계 프롬프트 정교화 (2023~2024): "잘 묻는 법"
- **3단계 자료·도구 통합 (2025~지금): "내 자료·내 도구를 AI 안으로"**

**B-2. 통합의 4가지 축 (2분)**
- 위 spine의 4축 슬라이드화 + 어느 차시에서 다루는지 매핑

**B-3. AI 도구 4가지 얼굴 (2분)**
| 도구 | 강점 | 오늘 다룰까 | 어디서 만날까 |
|---|---|---|---|
| ChatGPT | 데이터·엑셀 자동화 | ❌ | 별도 심화 과정 |
| Gemini | Google 생태계 통합 | ✅ 주력 | 1~4차시 전체 |
| Claude | 한국어 비즈니스 문체 | △ 보너스 | 1차시 보너스 |
| Perplexity | 출처 붙는 리서치 | △ 예고 | 2차시 보너스 |

### C 프롬프트 원칙 (8분) — ROADS 5원칙 (책 기반)

**책의 "프롬프트 마스터" 프레임을 도입**:

| 단계 | 내용 |
|---|---|
| **C** — Context (상황) | 어디에 활용하는가? |
| **O** — Objective (목적) | 무엇을 얻고 싶은가? |
| **C** — Constraints (제약) | 분량·형식·대상·기간 |
| **F** — Facts (참고 정보) | 데이터·배경·내부 상황 |
| **T** — Target (대상자) | 누가 결과물을 보는가? |

이 5단계가 기존 "역할·맥락·Few-shot 3원칙"을 흡수하면서 더 체계적인 프레임이 됨. 수강생이 외울 수 있는 약자 형태 (C.O.C.F.T 또는 ROADS).

### D 시연 (10분) — Tona Before/After

**시나리오: 신문광고 문안 작성 (오사카 단풍 5일 패키지)**

**Before 프롬프트 (1단계 채팅):**
```
일본 오사카 5일 패키지 신문광고 문안 써줘
```
→ 결과: "꿈에 그리던 오사카로의 환상적인 여행..." 같은 진부한 카피

**After 프롬프트 (Tona 5단계):**
```
Tona, 시작하자. 첫 번째 질문을 해줘.

(AI 질문 → 답변)
1. 문서 목적: 신문광고 문안
2. 독자: 조선일보 5단 광고 / 40~60대 부부
3. 길이: 메인 카피 1줄 + 서브 1줄 + 본문 80자, 3안 제시
4. 상단 요약: 없음
5. 소제목: 메시지형
6. 본문 스타일: 간결형 + 감성

내용:
- 상품: 오사카·교토 단풍 5일 (11월)
- 핵심: 사찰 단풍 명소 3곳 + 료칸 1박
- 가격: 169~189만원
- 강조: 단풍 시기 한정, 료칸 단독 객실
```
→ 결과: 회사 톤에 맞는 차별화된 카피 3안

### B @mention 시연 순서 (15분, 각 1~2분)
1. @Drive — "내 작년 일본 기획안 불러와"

5. @Calendar — "이 출장 일정 캘린더에 등록해줘"
6. @Tasks — "후속 조치 사항 Tasks에 추가해줘"
7. @Keep — "이 메모 Keep에 저장해줘"

### D 실습 — 메일 마스터 7단계 (15분)

책의 메일 마스터 프롬프트를 활용. 7단계 질문 순서:
1. To/CC (수신 정보)
2. 메일 목적
3. **수신자의 예상 질문** (★ 가장 강력한 포인트)
4. 본문 핵심 내용
5. 기대 행동
6. 톤 앤 매너 (격식+친절 / 협조+실무 / 간결+보고)
7. 첨부

**롯데관광 예시 시나리오**:
- 영업팀: 일본 랜드사에 가을 단풍 호텔 60실 추가 요청 (영문)
- 예약팀: 고객에게 항공 일정 변경 안내
- 경영지원: 임원에게 사내 워크숍 기안

### D 종합 자기선언 운영

수강생 워크북에 다음 4칸이 미리 배치되어 있음:
```
[1차시 체크인] 가장 인상 깊은 한 단어: ______
[2차시 체크인] 가장 써먹을 만한 한 가지: ______
[3차시 체크인] 부서에 가장 필요한 것: ______
[4차시 종합] 다음 주에 시도할 1가지: ______________________
```

- 0~3분: 4칸을 채우고 구글 폼 입력
- 3~10분: 부서별 1명씩 30초 발표 (7명 × 30초 = 3.5분 + 정리)
- 10~12분: 강사 마무리 + 자율 학습 로드맵 연결
