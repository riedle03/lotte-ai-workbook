# 09. 2차시 흐름 정리

> 차시 재설계 v2 — 책 4장 Tona + 강사 자작 "카피 마스터" 결합
> 세부 시연 시나리오는 별도 문서, 이 파일은 흐름만

---

## 2차시 핵심 메시지

> **"비즈니스 문서엔 비즈니스 톤이, 광고 카피엔 감성 톤이 필요하다.
> 책의 Tona는 전자, 새로 만든 '카피 마스터'는 후자를 담당한다.
> 두 마스터를 차례로 손에 익힌다."**

**손에 들고 나갈 산출물 2개**:
1. 본인 직군 **보고서·기획안 1건** (A4 1~3장, Tona)
2. 본인 직군 **광고 카피·홍보 문구 1건** (짧고 임팩트, 카피 마스터)

**핵심 도구**: Gemini + Tona (책 4장) + 카피 마스터 (강사 자작)

---

## 50분 블록 구성 (전반·후반 2부 구성)

| 블록 | 시간 | 핵심 |
|---|---|---|
| A. 오프닝 | 4분 | 1차시 회고 + "이번 차시는 두 가지 톤 — 보고서 톤 + 카피 톤" |
| **[전반] Tona — 보고서·기획안 톤** |
| B. Tona 원리·시연 | 11분 | 6단계 선택값 설명 + 기획안 Before/After 라이브 |
| C. Tona 실습 | 11분 | 본인 직군 보고서·기획안 1건 |
| **[후반] 카피 마스터 — 광고 카피 톤** |
| D. 카피 마스터 원리·시연 | 9분 | "Tona는 미사여구 제거, 카피는 정반대" + 신문광고 Before/After |
| E. 카피 마스터 실습 | 11분 | 본인 직군 카피 1건 (홍보·신문광고·기획전 카피 등) |
| F. 마무리 | 4분 | 체크인 + 3차시 예고 (NotebookLM·정보 통합) |

---

## A. 오프닝 (4분) — "이번 차시는 두 가지 톤"

1. 1차시 체크인 한두 명 호명 — "인상 깊은 한 단어" 공유 (1분)
2. 1차시·2차시 연결 메시지 (3분)
   - "1차시는 ROADS로 **뼈대**를 배웠다 (프롬프트 마스터)"
   - "이번 차시는 **톤·맛**을 배운다 — 단, 두 종류다"
   - 슬라이드 1장: 비즈니스 문서 톤 vs 광고 카피 톤 비교
     - 비즈 문서: 객관·중립·숫자·짧은 명사 → **Tona**
     - 광고 카피: 감성·임팩트·차별화·표현 → **카피 마스터**

---

## B. 기획 방향성 마스터 + Tona 원리·시연 (11분)

### 기획 방향성 마스터 도입 (3분)

- **도입 메시지** (1분): "Tona로 보고서를 쓰기 전에 — 무엇을 쓸지 먼저 정리해야 한다"
  - "기획 방향성 마스터 = 사고 코치. 생각 흐름을 질문으로 구조화해 목차까지 뽑아준다"
  - "기획 방향성 마스터(뭘 쓸지) → Tona(어떻게 쓸지) 2단계가 이번 차시 전반의 흐름"
- **라이브 시연** (2분): STEP 1 과제 기본 정보 → STEP 2 ASK 분석 → STEP 5 목차 자동 도출
  - 시연 시나리오: 가을 동남아 가족 기획전 콘셉트 기획안 (더미)
  - "이 목차를 Tona에 넘긴다"

### Tona 원리·시연 (8분)

1. **Tona 6단계 선택값 설명** (2분)
   - ①목적 ②독자 ③길이 ④상단 요약 여부 ⑤소제목 스타일 ⑥본문 스타일
   - 8대 원칙 (감정·미사여구 제거 / 숫자 병기 / 화살표 흐름 / "필요한 상황임" 결론)
   - 한 줄 정리: **"Tona = 회사 보고서·기획안의 톤 엔지니어"**

2. **Tona 라이브 시연** (6분)
   - Before: *"분기 매출 보고서 써줘"* → 두루뭉술
   - After: 기획 방향성 마스터 목차 → Tona 6단계 답변 → 회사 톤 보고서 출력
   - 시연 시나리오: 기획전 콘셉트 보고서 (가을 동남아 가족 기획전 더미)

---

## C. Tona 실습 (11분) — 본인 직군 보고서·기획안

워크북에 **기획 방향성 마스터 + Tona 전문 인쇄본** 제공 (강사가 직접 제공)

> **실습 2단계 흐름**: 기획 방향성 마스터로 목차 먼저 → Tona로 보고서 포맷


| 직군 | Tona 실습 산출물 |
|---|---|
| 상품기획 | 신상품 기획안 |
| 영업 | 거래선 협의 결과 보고서 |
| 예약·발권 | 분기 예약 현황 보고 |
| 경영지원·인사총무 | 부서 운영 보고 / 사업 제안 |
| 고객서비스·VOC | VOC 월간 분석 보고 |
| 카지노·VIP | VIP 등급별 운영 보고 |
| IT·시스템 | 시스템 운영 현황 보고 |

---

## D. 카피 마스터 원리·시연 (9분)

1. **카피 마스터 도입 메시지** (2분)
   - "Tona는 미사여구 제거가 원칙이었다. 카피는 정반대 — 감성·임팩트·차별화"
   - "그래서 책의 Tona 그대로는 카피에 안 맞다. 강사가 카피용 마스터를 별도로 만들었다"
   - 카피 마스터 = ROADS 원리 + 광고 카피 특화 질문 구조

2. **카피 마스터 라이브 시연** (7분)
   - Before: *"오사카 단풍 신문광고 카피 써줘"* → 진부한 "꿈에 그리던…"
   - After: 카피 마스터 질문에 답변 → 차별화된 메인·서브·본문 3안
   - 시연 시나리오: 신문광고 (오사카 단풍 5일 더미)

---

## E. 카피 마스터 실습 (11분) — 본인 직군 카피

| 직군 | 카피 마스터 실습 산출물 |
|---|---|
| 상품기획 | 신상품 홍보 카피 / 기획전 카피 |
| 영업 | 거래선용 세일즈 한 줄 |
| 예약·발권 | 고객용 프로모션 안내 카피 |
| 경영지원·인사총무 | 사내 캠페인 슬로건 |
| 고객서비스·VOC | 감사 카드 한 줄 |
| 카지노·VIP | VIP 이벤트 초청 카피 |
| IT·시스템 | (해당 시 시스템 런칭 카피) / 또는 Tona로 시간 확장 |

**숙련자 보너스**:
- 같은 카피를 Claude로 다시 → 한국어 톤 차이 비교
- GPT로 같은 카피의 캘리그라피·이미지 변환 (회신에서 언급된 이미지 수요)

---

## F. 마무리 (4분)

- **체크인**: "이번 차시에서 가장 써먹을 만한 한 가지" 옆 사람과 1분 + 워크북 2번 칸 기록
- **3차시 예고**: "지금까지는 내 머릿속·외부 자료로 만들었다. 다음은 **내 부서 자료**를 AI에게 학습시키는 시간 — NotebookLM"

## 메시지 누적 흐름 (한눈에)

```
A. 1차시는 뼈대(ROADS) → 이번 차시는 톤·맛 (두 종류)
        ↓
B. 기획 방향성 마스터 — "무엇을 쓸지" 사고 정리 + 목차 도출
        ↓
B. Tona 원리 — "어떻게 쓸지" 보고서·기획안 톤 엔지니어 (객관·중립·숫자)
        ↓
B. 2단계 통합 시연 — 기획전 콘셉트 기획안 Before/After
        ↓
C. 본인 직군 보고서·기획안 실습 (기획 방향성 마스터 → Tona 순서)
        ↓
D. 카피 마스터 — Tona와 정반대 톤 (감성·임팩트·차별화)
        ↓
D. 카피 마스터 시연 — 신문광고 Before/After (오사카 단풍)
        ↓
E. 본인 직군 카피·홍보 문구 실습
        ↓
F. 써먹을 만한 한 가지 + 3차시(NotebookLM) 예고
```

## 결정 필요한 항목

1. **카피 마스터 프롬프트** — 강사 자작 필요
   - 강사가 직접 작성 or 클로드 코드에서 초안 작성 후 검수
2. **2차시 전반·후반 사이 전환 1분** — 휴식 없이 50분 안에 두 마스터 다 다루는 부담
   - 강사 워밍업 멘트 / 슬라이드 전환 신호 필요
3. **카피 마스터 시연·실습 시나리오** — 책의 광고 카피 사례가 부족하므로 강사 더미 시나리오 필요
   - 신문광고 (오사카 단풍) / 기획전 카피 (가을 동남아 가족) 등
