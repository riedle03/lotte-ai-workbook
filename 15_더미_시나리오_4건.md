# 15. 더미 시나리오 4건

> 롯데관광개발 검수 요청용 — 강의 시연·실습에 실제 사용할 더미 데이터 + 기대 산출물 샘플
> 실데이터 없음. 모두 가상 설정.
> 회신 7가지 업무 ①~⑦ 중 핵심 4개 커버.

## 시나리오 1 — 신문광고 카피 (업무 ③)

> **해당 업무**: 기사광고·신문광고 문안 작성  
> **사용 프롬프트**: 카피 마스터 (2차시 D·E 블록 대표 시연)  
> **도구**: Gemini

### 더미 상품 설정

| 항목 | 내용 (더미) |
|---|---|
| 상품명 | 오사카·교토 단풍 완전정복 5일 |
| 출발일 | 2026년 11월 7일(토) |
| 가격 | 1,290,000원 (2인 기준, 항공 포함) |
| 항공 | 인천 → 간사이 / 대한항공 |
| 호텔 | 오사카 시내 4성급 2박 + 교토 3성급 2박 |
| 핵심 포인트 | 나라공원 야간 단풍 야간 개방 포함 / 오사카 도톤보리 자유시간 3시간 / 교토 아라시야마 새벽 입장 (인파 없음) |
| 모집 인원 | 선착순 28명 |
| 광고 매체 | 조선일보 여행 섹션 (A4 1/4면, 흑백) |

### 카피 마스터 입력값 (더미)

```
① 상품/서비스: 오사카·교토 단풍 완전정복 5일 패키지
② 핵심 메시지 1줄: 인파 없는 새벽 교토, 야간 나라 단풍 — 남들과 다른 단풍 여행
③ 타깃: 40~50대 직장인 부부, 1년에 한 번 여행, 사진 좋아함
④ 카피 유형: 신문광고 (조선일보 여행 섹션, 흑백)
⑤ 분량: 메인 카피 1줄 + 서브 카피 1줄 + 본문 3~4줄
⑥ 금기 표현: "꿈에 그리던", "환상적인", "최고의", 감탄사
⑦ 참고 예시: 없음
```

### 기대 산출물 예시 (AI 생성 샘플)

**[안 1 — 시간 차별화]**
> **메인**: 교토의 단풍은 새벽 6시에 시작된다
> **서브**: 아라시야마 새벽 전세 입장 · 나라 야간 단풍 포함
> **본문**: 관광객이 몰리기 전 교토 아라시야마를 걷는다. 해 질 녘 나라공원 단풍 야간 개방까지 — 같은 일본, 다른 일정. 11월 7일 출발 · 선착순 28명.

**[안 2 — 희소성]**
> **메인**: 단풍 시즌, 28명만 아는 일본이 있다
> **서브**: 새벽 아라시야마 + 야간 나라공원 · 인파 제로
> **본문**: 도톤보리 자유시간 3시간, 교토 료칸 2박, 나라 야간 단풍 개방 포함. 조선일보 독자 한정 특가 1,290,000원.

**[안 3 — 감도 중심]**
> **메인**: 빨갛기 전에 가야 가장 빨간 단풍을 본다
> **서브**: 11월 7일 출발 · 교토·오사카·나라 5일
> **본문**: 단풍 피크 직전 출발, 가장 짧은 대기, 가장 넓은 자유시간. 롯데관광 단풍 시즌 한정 패키지.

## 시나리오 2 — 기획전 콘셉트 + 카피 (업무 ⑤)

> **해당 업무**: 기획전 콘셉트 및 카피 작성  
> **사용 프롬프트**: 기획 방향성 마스터 → Tona (콘셉트 기획안) + 카피 마스터 (카피)  
> **도구**: Gemini  
> **강의 배치**: 2차시 B·C 블록 통합 시연 시나리오

### 더미 기획전 설정

| 항목 | 내용 (더미) |
|---|---|
| 기획전명 | 2026 가을 가족 동남아 기획전 |
| 기간 | 2026년 9월 1일~30일 (출발 기준) |
| 타깃 | 초등학생 자녀 둔 가족 (3~4인) |
| 목적 | 가을 비수기 매출 보완 + 가족 패키지 신규 수요 발굴 |
| 대상 상품 | 세부·보라카이·다낭·방콕 가족 패키지 (4개 노선) |
| 특전 | 어린이 항공·호텔 할인 50% + 키즈 액티비티 1종 무료 |
| 채널 | 롯데관광 홈페이지 메인 배너 + 이메일 뉴스레터 |

### 기획 방향성 마스터 입력값 (더미)

```
STEP 1
1️⃣ 지시자: 상품기획팀장
2️⃣ 최종 의사결정자: 상품본부장
3️⃣ 과제명: 2026 가을 가족 동남아 기획전 기획안

STEP 1-보강
Q. 추진 이유: 9~10월 가족 패키지 예약이 전년 대비 23% 감소. 경쟁사 대비 가족 전용 상품 부족이 원인으로 분석됨.

STEP 2 ASK
Attitude: ②성과·효율 중심, ③리스크 관리 중심
Style: ①결론 먼저, ②숫자·지표 중심
Knowledge: ①이슈에 익숙함
```

### Tona 기획안 기대 산출물 예시

```
■ 2026 가을 가족 동남아 기획전 기획안 (안)

[핵심 요약]
→ 9~10월 가족 패키지 예약 전년 대비 23% 감소 대응
→ 4개 노선 한정 특전 구성으로 가족 신규 수요 발굴
→ 예약 목표: 기획전 기간 내 100가족 (전년 동기 대비 +30%)

□ 추진 배경
 ○ (현황) 2026년 9~10월 가족 패키지 예약률 전년比 -23%
 ○ (원인) 경쟁사 대비 가족 전용 특전 부재 · 비수기 가격 메리트 미흡
 ○ (기회) 초등학생 자녀 가족의 추석 전후 여행 수요는 유지 중

□ 기획전 구성안
 ○ 대상 노선: 세부·보라카이·다낭·방콕 (4개)
 ○ 특전 구조
  - 어린이(만 12세 이하) 항공·호텔 50% 할인
  - 키즈 액티비티 1종 무료 (스노클링 또는 쿠킹클래스)
  - 3인 이상 예약 시 단체 할인 추가 적용

□ 채널 계획
 ○ 롯데관광 홈페이지 메인 배너 (9.1 기준 교체)
 ○ 이메일 뉴스레터 (전체 회원 발송 · 8.25 예정)

□ 기대 효과
 ○ 기획전 기간 예약 100가족 달성 시 매출 약 +1.8억 원 추정
 ○ 가족 고객 DB 확보 → 겨울 시즌 재구매 연결 가능성 높음

□ 리스크
 ○ 항공 좌석 블록 미확보 시 특전 제공 차질 가능 → 9.1 이전 항공사 협의 필요한 상황임
```

### 카피 마스터 기대 산출물 예시

**[안 1 — 아이 중심]**
> **메인**: 아이가 신나면, 엄마 아빠도 신난다
> **서브**: 어린이 50% · 키즈 액티비티 무료 · 4개 노선 선택
> **본문**: 세부·보라카이·다낭·방콕. 가을에만 열리는 가족 전용 특가. 9월 출발 한정.

**[안 2 — 비용 중심]**
> **메인**: 넷이 가는 여행, 둘 가격으로
> **서브**: 어린이 항공+호텔 반값 · 2026 가을 가족 기획전
> **본문**: 초등학생 이하 50% 할인 + 키즈 액티비티 1종 무료. 롯데관광 가을 시즌 한정.

**[안 3 — 추억 중심]**
> **메인**: 이번 추석, 아이와 바다 앞에 서자
> **서브**: 세부·보라카이·다낭·방콕 가족 패키지 · 9월 특가
> **본문**: 아이 항공·호텔 반값. 스노클링 또는 쿠킹클래스 무료 포함. 한 가족이 기억하는 가을.

## 시나리오 3 — 그룹웨어 기안문 (업무 ④)

> **해당 업무**: 그룹웨어 기안문 초안 작성  
> **사용 프롬프트**: 프롬프트 마스터 (1차시 D·E 블록 강사 시연 시나리오)  
> **도구**: Gemini

### 더미 기안문 설정

| 항목 | 내용 (더미) |
|---|---|
| 기안 부서 | 경영지원팀 |
| 기안 제목 | 2026 하반기 팀장급 리더십 워크숍 개최 |
| 일시 | 2026년 7월 18일(금)~19일(토) 1박 2일 |
| 장소 | 강원도 원주 오크밸리 리조트 (더미) |
| 대상 | 팀장급 이상 28명 |
| 예산 | 총 14,000,000원 (1인당 500,000원) |
| 목적 | 하반기 전략 공유 + 팀 간 협업 체계 강화 |
| 주요 일정 | 1일차: 전략 공유 세션 → 저녁 팀빌딩 / 2일차: 워크숍 결과 공유 → 귀환 |

### 프롬프트 마스터 입력값 (더미)

```
1️⃣ Context: 그룹웨어 기안문 작성 — 하반기 팀장급 워크숍 개최 승인 요청
2️⃣ Objective: 대표이사 결재 받을 수 있는 기안문 초안 자동 생성
3️⃣ Constraints: 사내 기안 양식 준수 / 예산 근거 포함 / A4 1장 이내
4️⃣ Facts: 7월 18~19일 / 강원 원주 오크밸리 / 팀장급 28명 / 총 14,000,000원 / 하반기 전략 공유 + 협업 강화 목적
5️⃣ Target: 대표이사 (결재권자), 경영지원팀장 (초안 작성 지시자)
```

### 기대 산출물 예시

```
기  안  문

기안일자: 2026. 07. 01.
기안부서: 경영지원팀
결  재: 팀장 → 본부장 → 대표이사

제    목: 2026년 하반기 팀장급 리더십 워크숍 개최의 건

1. 목적

   1박 2일 리더십 워크숍을 개최하고자 하오니 결재하여 주시기 바랍니다.

   □ 일  시: 2026. 7. 18.(금) ~ 7. 19.(토) / 1박 2일

   □ 대  상: 팀장급 이상 28명
   □ 주요내용
    ○ 1일차: 하반기 전략 방향 공유 세션 → 팀빌딩 프로그램

   □ 총 소요 예산: 14,000,000원
    ○ 숙박·식음: 9,800,000원 (1인 350,000원 × 28명)
    ○ 프로그램 운영: 2,800,000원 (1인 100,000원 × 28명)
    ○ 교통비: 1,400,000원 (1인 50,000원 × 28명)
   □ 예산 출처: 2026년 교육훈련비 기집행 예산 내 처리

4. 기대 효과
   ○ 하반기 팀 간 전략 방향 정렬 → 실행력 제고
   ○ 팀장급 네트워크 강화 → 부서 간 협업 효율 향상 기대

붙임: 세부 일정표 1부. 끝.
```
