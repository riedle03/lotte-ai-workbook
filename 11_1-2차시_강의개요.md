# 1차시 (50분) — "AI 신입사원이 들어왔다, 우리는 무엇을 준비해야 하나"

## 핵심 메시지

> **"AI는 증폭기다. 좋은 역량도 10배, 쓰레기도 10배.
> 증폭기의 원료는 '양질의 데이터'와 '잘 설계된 질문'이다."**

**산출물**: 본인 직군 **정형 비즈니스 문서 1건** (A4 1장 이하 — 메일·기안·안내·공지)
**핵심 도구**: Gemini + 프롬프트 마스터 (책 1·2장)

## 50분 블록

| 블록 | 시간 | 핵심 |
|---|---|---|
| A1. 오프닝 | 5분 | 중앙일보 기사(퀘스처닝 시대) → AI 신입사원 비유 |
| A2. AI 도구 지형도 | 5분 | 5도구 소개 + 보안 환기 + "오늘은 Gemini·NotebookLM 중심" |
| B. 자기 진단·메시지 | 8분 | 4유형 진단 → 증폭기 → 양질의 데이터 + 7업무 매핑 |
| C. 프롬프트 원칙 | 7분 | ROADS 5원칙 (역·목·길·단·다) |
| D. 시연 | 10분 | 프롬프트 마스터 라이브 (ROADS가 실제 작동) |
| E. 실습 | 12분 | 본인 직군 시나리오로 프롬프트 마스터 실습 |
| F. 마무리 | 3분 | 체크인 + 2차시 예고 |

## 메시지 흐름

```
A1. 같은 AI 써도 결과 다른 이유 = 질문 능력 (중앙일보)
        ↓
A1. AI 신입사원 비유 — 사고력이 사라진다 (책)
        ↓
A2. 5개 도구 지형도 — Gemini·NotebookLM 중심 / GPT·Perplexity·Claude는 호명
        ↓
B. 4유형 자기 진단 → 사고 설계형으로 가자
        ↓
B. AI는 증폭기 — 좋은 결과도 10배, 쓰레기도 10배
        ↓
B. 증폭기 원료 = 양질의 데이터 (7업무 매트릭스)
        ↓
C. 설계도가 필요하다 → ROADS 5원칙
        ↓
C→D 다리. 매번 직접 짜기 힘드니까 → 프롬프트 마스터(ROADS 통역사)
        ↓
D. 프롬프트 마스터 라이브 시연
        ↓
E. 본인 직군 실습 + 숙련자 보너스(메일 마스터 비교)
        ↓
F. "인상 깊은 한 단어" 체크인 + 2차시(Tona) 예고
```

# 2차시 (50분) — "회사답게 만드는 법: 두 가지 톤"

## 핵심 메시지

> **"비즈니스 문서엔 비즈니스 톤이, 광고 카피엔 감성 톤이 필요하다.
> 책의 Tona는 전자, 자작 카피 마스터는 후자를 담당한다."**

**산출물 2개**:
1. 본인 직군 **보고서·기획안 1건** (A4 1~3장, Tona)
2. 본인 직군 **광고 카피·홍보 문구 1건** (짧고 임팩트, 카피 마스터)

**핵심 도구**: Gemini + Tona (책 4장) + 카피 마스터 (강사 자작)

## 50분 블록 (전반·후반 2부 구성)

| 블록 | 시간 | 핵심 |
|---|---|---|
| A. 오프닝 | 4분 | 1차시 회고 + "두 가지 톤 — 보고서 톤 + 카피 톤" |
| **[전반] Tona — 보고서·기획안 톤** |
| B. Tona 원리·시연 | 11분 | 6단계 선택값 + 기획안 Before/After |
| C. Tona 실습 | 11분 | 본인 직군 보고서·기획안 1건 |
| **[후반] 카피 마스터 — 광고 카피 톤** |
| D. 카피 마스터 원리·시연 | 9분 | "Tona는 미사여구 제거, 카피는 정반대" + 신문광고 Before/After |
| E. 카피 마스터 실습 | 11분 | 본인 직군 카피 1건 (홍보·신문광고·기획전) |
| F. 마무리 | 4분 | 체크인 + 3차시 예고 (NotebookLM) |

## 메시지 흐름

```
A. 1차시는 뼈대(ROADS) → 이번 차시는 톤·맛 (두 종류)
        ↓
B. Tona 원리 — 보고서·기획안 톤 엔지니어 (객관·중립·숫자)
        ↓
B. Tona 시연 — 기획전 콘셉트 보고서 Before/After
        ↓
C. 본인 직군 보고서·기획안 실습
        ↓
D. 카피 마스터 — Tona와 정반대 톤 (감성·임팩트·차별화)
        ↓
D. 카피 마스터 시연 — 신문광고 Before/After (오사카 단풍)
        ↓
E. 본인 직군 카피·홍보 문구 실습
        ↓
F. "써먹을 만한 한 가지" 체크인 + 3차시(NotebookLM) 예고
```
