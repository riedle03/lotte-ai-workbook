# 16. 직군별 실습 카드 (7직군 × 4차시)

> 각 카드 구성: 상황 설정 → 프롬프트 입력 대화 예시 → 산출물 예시
> 책 사례 형식을 롯데관광 업무에 맞춰 작성
> 모든 데이터는 더미 (실데이터 아님)

# 1차시 실습 카드 — 프롬프트 마스터

> **흐름**: Gemini에 프롬프트 마스터 붙여넣기 → 5단계 질문 답변 → 실행 프롬프트 자동 생성 → 결과 생성

## [1-1] 상품기획·오퍼레이터

**상황**: 11월 출시 예정인 오사카 단풍 패키지를 담당 팀장에게 내부 공유하는 런칭 공지문 필요

**프롬프트 마스터 입력**
```
1️⃣ Context: 신상품 내부 공지 — 오사카 단풍 패키지 11월 출시 예정
2️⃣ Objective: 상품 개요·특장점·판매 시작일을 팀원들에게 간결하게 공유하는 내부 공지문
3️⃣ Constraints: A4 반 장 이내 / 불릿 형식 / 과장 표현 없이
4️⃣ Facts: 오사카·교토 5일 / 11월 7일 출발 / 1,290,000원 / 선착순 28명 / 새벽 아라시야마 포함
5️⃣ Target: 영업팀·예약팀 실무자 (내부 공지 수신자)
```

**산출물 예시**
```
[내부 공지] 2026년 하반기 신상품 런칭 안내

■ 상품명: 오사카·교토 단풍 완전정복 5일
■ 출발일: 2026. 11. 7.(토)
■ 판매가: 1,290,000원 (2인 기준, 항공 포함)
■ 모집: 선착순 28명

□ 주요 특장점
 ○ 교토 아라시야마 새벽 입장 — 인파 없는 단풍 감상
 ○ 나라공원 야간 단풍 개방 포함
 ○ 오사카 도톤보리 자유시간 3시간

□ 판매 시작: 2026. 9. 1.(화) 오전 10시
□ 문의: 상품기획팀 내선 ○○○○

조기 마감 예상 상품으로 영업·예약팀 선판매 대응 요청드립니다.
```

## [1-2] 영업·판매

**상황**: 일본 현지 랜드사(K-Land)에 2026 가을 단풍 시즌 호텔 블록 추가를 요청하는 영문 비즈니스 메일

**프롬프트 마스터 입력**
```
1️⃣ Context: 일본 현지 랜드사에 호텔 블록 추가 요청 영문 메일
2️⃣ Objective: 11월 단풍 시즌 오사카 호텔 60실 추가 블록 확보 요청
3️⃣ Constraints: 영문 / 격식+친절 / 회신 기한 명시 / A4 반 장
4️⃣ Facts: 수신 T.Yamamoto@k-land.co.jp / 60실 / 11.1~15 / 결제 조건 기존 동일 / 회신 5.30까지
5️⃣ Target: 일본 랜드사 담당자 (기존 거래처)
```

```
Best regards,
[Name] | Sales Team, Lotte Tour Development
```

## [1-3] 예약·발권

**상황**: 고객에게 항공편 일정 변경(운항 취소로 인한 대체편 안내) 공지 메일 발송

**프롬프트 마스터 입력**
```
1️⃣ Context: 항공편 운항 취소로 인한 대체편 안내 고객 메일
2️⃣ Objective: 불안을 최소화하면서 대체편 일정과 환불/변경 옵션을 명확하게 안내
3️⃣ Constraints: 고객용 / 친절+명확 / 사과 포함 / 행동 요청 명시
4️⃣ Facts: 원래 편 OZ101 11.7 10:00 → 대체편 OZ103 11.7 15:30 / 환불 또는 대체편 중 선택 / 선택 기한 5.28 18시
5️⃣ Target: 해당 상품 예약 고객 (일반 소비자)
```

**산출물 예시**
```
제목: [롯데관광] 항공편 일정 변경 안내 — 고객님 예약 건

고객님, 안녕하세요. 롯데관광개발 예약팀입니다.

고객님의 여행 일정 관련 중요한 안내 말씀을 드립니다.

■ 변경 내용
 - 기존: OZ101편 / 11월 7일(토) 10:00 출발
 - 변경: OZ103편 / 11월 7일(토) 15:30 출발 (운항사 사정으로 변경)

■ 선택 가능 옵션 (5월 28일(목) 오후 6시까지 회신 부탁드립니다)
 ① 대체편(OZ103) 그대로 탑승 — 추가 비용 없음
 ② 전액 환불 — 영업일 기준 5일 이내 처리

불편을 드려 진심으로 사과드립니다.
문의: 예약팀 직통 02-○○○-○○○○ (평일 09:00~18:00)
```

## [1-5] 고객서비스·VOC

**상황**: 여행 중 호텔 룸 컨디션 불만을 제기한 고객에게 귀국 후 사후 처리 결과 안내 메일

**프롬프트 마스터 입력**
```
1️⃣ Context: 고객 불만(호텔 룸 컨디션) 사후 처리 결과 안내 메일
2️⃣ Objective: 사과 + 처리 결과 + 보상 내용을 전달하며 관계 회복
3️⃣ Constraints: 진심 어린 톤 / 보상 내용 명확히 / 재방문 유도 포함
4️⃣ Facts: 불만 내용=욕실 곰팡이·냉방 불량 / 처리=현지 호텔 측에 공식 항의 완료 / 보상=다음 예약 시 3만 원 할인 쿠폰
5️⃣ Target: 불만 고객 (기존 이용자)
```

**산출물 예시**
```
제목: [롯데관광] 불편 사항 처리 결과 및 사과 말씀

○○○ 고객님, 안녕하세요. 롯데관광개발 고객서비스팀입니다.

여행 중 불편을 겪으셨음에도 끝까지 함께해 주셔서 감사드립니다.

■ 처리 결과
고객님께서 제기하신 욕실 시설 및 냉방 관련 불편 사항을
현지 호텔 측에 공식 전달하여 시설 점검 및 개선 조치를 요청하였습니다.
향후 동일 상품 이용 고객에게는 재발하지 않도록 관리할 것을 약속드립니다.

■ 보상 안내
사과의 마음을 담아 다음 여행 예약 시 사용 가능한
30,000원 할인 쿠폰을 발송해 드렸습니다. (유효기간: 발급일로부터 1년)

앞으로도 더 나은 서비스로 보답하겠습니다.
```

# 2차시 전반 실습 카드 — 기획 방향성 마스터 → Tona

> **흐름**: 기획 방향성 마스터 STEP 1~5 → 목차 도출 → Tona 6단계 선택 → 보고서 완성

## [2A-1] 상품기획·오퍼레이터

**상황**: 2026 가을 동남아 가족 기획전 기획안 작성 (본부장 보고용)

> ※ 2차시 B블록 강사 시연 시나리오

**기획 방향성 마스터 핵심 입력**
```
과제명: 2026 가을 가족 동남아 기획전 기획안
의사결정자: 상품본부장
ASK: 성과·효율 중심 / 결론 먼저 / 숫자·지표 중심
추진 이유: 9~10월 가족 패키지 예약 전년比 -23% 대응
```

**목차 도출 결과**
```
① 왜 지금 가족 기획전인가 (추진 배경)
② 현재 시장 상황은 (경쟁사·수요 동향)
③ 우리 상품의 문제는 무엇인가 (현황 진단)
④ 기획전 구성안 (노선·특전·기간)
⑤ 실행 계획 (채널·일정)
⑥ 기대 효과 및 리스크
```

**Tona 선택값**
```
목적: 기획 / 독자: 본부장 / 길이: 2페이지
상단 요약: 상단 헤드메시지형 / 소제목: 포괄 메시지형 / 본문: 전략메시지형
```

→ 산출물 예시: 시나리오 2 (15번 파일) Tona 기획안 참조

## [2A-2] 영업·판매

**상황**: 신규 동남아 랜드사 계약 타당성 검토 보고서 (팀장 보고용)

**기획 방향성 마스터 핵심 입력**
```
과제명: 베트남 다낭 신규 랜드사 ○○Travel 계약 타당성 검토
의사결정자: 영업팀장
ASK: 리스크 관리 중심 / 비교·대안 구조 선호 / 이슈 익숙함
추진 이유: 기존 랜드사 서비스 품질 저하로 대체 파트너 필요
```

**목차 도출 결과 + Tona 보고서 예시**
```
[상단 요약]
→ 기존 랜드사 VOC 3분기 연속 증가 → 대체 파트너 발굴 필요성 높음
→ 신규 ○○Travel 현장 실사 및 견적 비교 완료
→ 계약 추진 권고 (단, 계약 조건 2항 수정 필요한 상황임)

■ "기존 파트너사 서비스 품질이 하락하고 있다"
 ○ (현황) 다낭 노선 VOC 불만 건수: 1Q 8건 → 2Q 14건 → 3Q 21건 (+162%)
 ○ (원인) 기존 랜드사 현지 가이드 이직률 급증 / 차량 노후화 미개선

■ "신규 파트너 ○○Travel 비교 검토 결과"
 ○ 가격: 기존 대비 +3.2% / 서비스 품질 현장 실사 B+ 평가
 ○ 리스크: 계약 해지 위약금 조항 기존보다 불리 → 재협의 필요한 상황임

■ "계약 추진 권고 및 요청 사항"
 ○ 계약 조건 2항(위약금) 수정 협의 후 최종 승인 요청
```
