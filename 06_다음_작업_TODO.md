## 작업 환경 메모

### 클로드 코드 사용 팁

1. **CLAUDE.md가 자동 로드**되므로 매번 컨텍스트 설명 안 해도 됨
2. 새 자료 추가 시 **CLAUDE.md의 "파일 가이드" 섹션 업데이트**
3. 작업 완료 항목은 **이 TODO 파일에서 체크**
4. docx 생성은 `outputs/` 폴더에, 작업 중간 자료는 `cuts/` 폴더에

### 폴더 구조 권장

```
lotte-tour-ai-lecture/
├── CLAUDE.md                    ← 마스터 (자동 로드)
├── 01_강의_전체_설계.md
├── 02_설문_분석_데이터.md
├── 03_롯데관광_회신_및_업무.md
├── 04_프롬프트_원본_3종.md
├── 05_책_22개_항목_분류.md
├── 06_다음_작업_TODO.md         ← 이 파일
│
├── cuts/                        ← 작업 중간 산출물
│   ├── 01_시연_시나리오/
│   ├── 02_더미_시나리오/
│   ├── 03_직군별_시나리오/
│   ├── 04_차시별_개요/
│   └── 05_사전_안내_메일/
│
└── outputs/                     ← 최종 산출물 (docx 등)
    ├── 강의계획서_v2.docx
    ├── 사전_안내_메일.docx
    └── 디지털_자료_패키지/
```
