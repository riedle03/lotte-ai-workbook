# 항공권 변경·환불 처리 업무 매뉴얼

**문서번호**: RES-OPS-2025-03
**작성부서**: 예약·발권팀
**최종수정일**: 2025년 5월
**적용시스템**: ResyPro v4.2

---

## 1. 목적

본 매뉴얼은 롯데관광개발 예약·발권팀 소속 직원이 항공권 변경 및 환불 요청을 처리할 때 일관된 절차와 기준을 따르도록 하여 고객 서비스 품질을 유지하고 업무 오류를 최소화하는 것을 목적으로 한다.

---

## 2. 적용 범위

- 롯데관광개발이 발권한 모든 국제선·국내선 항공권
- 개인 예약 및 단체 예약(10인 이상) 포함
- 직접 발권분 및 GDS(Global Distribution System) 경유 발권분 모두 해당

---

## 3. 용어 정의

| 용어 | 정의 |
|---|---|
| **변경(Reissue)** | 출발일, 구간, 편명 등 항공권 조건을 수정하는 처리 |
| **환불(Refund)** | 발권된 항공권을 취소하고 요금을 돌려주는 처리 |
| **노쇼(No-show)** | 사전 취소 없이 해당 항공편에 탑승하지 않은 경우 |
| **페널티(Penalty)** | 변경·환불 시 항공사 규정에 따라 공제되는 수수료 |
| **Void** | 발권 당일 자정 이전에 티켓을 취소하는 것 (페널티 없음) |
| **ResyPro** | 롯데관광개발 사내 예약·발권 통합 관리 시스템 |

---

## 4. 변경 처리 절차

### 4-1. 출발 72시간 이전 변경

- **수수료**: 없음 (단, 항공사별 운임 차액은 고객 부담)
- **처리 방법 (ResyPro 기준)**:
  1. ResyPro 메인 화면 → [예약조회] → 예약번호(PNR) 입력
  2. [항공 상세] 탭 → [변경 요청] 버튼 클릭
  3. 변경 사유 선택: `고객 요청 / 일정 조정 / 항공사 스케줄 변경` 중 택 1
  4. 변경 내용(날짜·편명·구간) 입력 후 [항공사 조회] 클릭 → 재고 확인
  5. 운임 차액 자동 계산 확인 → 고객에게 안내 후 [승인 요청] 전송
  6. 항공사 승인 수신 → [발권 처리] → 새 티켓번호 생성 확인
  7. 고객에게 변경 확인서 이메일 자동 발송 (ResyPro [알림] 탭에서 확인)

### 4-2. 출발 72시간 이내 변경

- **수수료**: 항공사별 규정 적용 (아래 참고)
  - 대한항공(KE): 편당 USD 50~100 (클래스별 상이)
  - 아시아나항공(OZ): 편당 USD 50~80
  - 저비용항공사(LCC): 원칙적으로 변경 불가 또는 전액 몰취
- **처리 방법**: 4-1과 동일하되, 3단계에서 [72시간 이내] 체크박스 활성화 → 페널티 자동 산출 확인 후 고객 동의서 첨부 필수

---

## 5. 환불 처리 절차

환불 처리는 아래 6단계를 순서대로 진행하며, **평균 처리 기간은 3~5 영업일**이다.

| 단계 | 내용 | 담당 |
|---|---|---|
| **1. 고객 요청 접수** | 고객이 전화·이메일·채팅으로 환불 요청 → ResyPro [환불접수] 탭에 요청 내용 입력 | 담당 직원 |
| **2. 항공사 확인** | PNR 조회 → 항공사 규정 및 잔여 유효기간 확인 (ResyPro [항공사 연동] 자동 조회) | 담당 직원 |
| **3. 수수료 안내** | 페널티·환불 가능 금액 계산 후 고객에게 구두 또는 이메일로 안내 | 담당 직원 |
| **4. 고객 승인** | 고객이 환불 조건에 동의 → 동의 확인서(ResyPro 자동 생성 양식) 이메일 회신 수령 | 담당 직원 |
| **5. 환불 처리** | ResyPro [환불 실행] → 항공사 환불 요청 전송 → 처리번호(RF번호) 저장 | 담당 직원 |
| **6. 완료 통보** | 환불 완료 후 ResyPro 자동 알림으로 고객에게 완료 이메일 발송, 처리일·금액 명기 | 시스템 자동 |

> **주의**: 신용카드 환불의 경우 카드사 처리 일정에 따라 실제 계좌 반영까지 추가 3~7 영업일이 소요될 수 있음을 고객에게 사전 안내한다.

---

## 6. 노쇼(No-show) 처리 기준

- 노쇼 확인: 출발 후 2시간 이내 ResyPro [탑승 현황] 탭에서 자동 플래그 표시
- **노쇼 발생 시 환불 원칙**:
  - 일반 운임(Y/B 클래스): 세금 및 유류할증료 일부만 환불 (항공 운임은 몰취)
  - 저렴 운임(V/Q 클래스 이하): 전액 몰취 (환불 불가)
  - 비즈니스·퍼스트 클래스: 항공사별 별도 규정 확인 후 처리
- 노쇼 후 귀국편이 연동된 경우 자동 취소 여부를 반드시 항공사에 확인한다.
- ResyPro에서 [노쇼 처리] 버튼 클릭 → 사유 코드 `NS` 입력 → 기록 저장

---

## 7. 단체(10인 이상) vs 개인 처리 기준 차이

| 구분 | 개인 예약 | 단체 예약 (10인 이상) |
|---|---|---|
| **변경 권한** | 담당 직원 단독 처리 가능 | 팀장 사전 승인 필요 |
| **환불 처리** | 즉시 접수·처리 | 단체 담당자 공문 필요 |
| **페널티 적용** | 개인별 적용 | 단체 계약 조건 우선 적용 |
| **처리 기간** | 3~5 영업일 | 5~10 영업일 (항공사 협의 필요) |
| **ResyPro 경로** | [개인 예약] 메뉴 | [단체 예약] 메뉴 → [그룹 PNR] 선택 |
| **환불 수령인** | 결제 고객 본인 | 대표 계약자 또는 법인 계좌 |

---

## 8. 유의사항

1. **Void 처리 우선 확인**: 발권 당일이라면 환불이 아닌 Void로 처리해야 페널티를 방지할 수 있다. ResyPro에서 발권 시각과 현재 시각을 반드시 비교한다.
2. **이중 처리 방지**: ResyPro에서 환불 요청 전송 후 반드시 [처리 현황] 탭에서 중복 요청 여부를 확인한다.
3. **외화 환불**: 외화로 발권된 티켓의 환불은 발권 당시 환율이 아닌 환불 처리일 환율 기준으로 정산될 수 있음을 고객에게 고지한다.
4. **항공사 직접 연락 금지**: 고객이 항공사에 직접 환불을 요청하면 여행사 PNR이 분리될 수 있으므로, 반드시 예약·발권팀을 통해 처리하도록 안내한다.

## 9. 자주 발생하는 오류 케이스 3가지

### 오류 1: ResyPro에서 "항공사 응답 없음" 오류 발생

- **증상**: [항공사 조회] 클릭 후 30초 이상 응답 없거나 `ERR-GDS-504` 코드 표시
- **원인**: GDS 연동 서버 일시 장애 또는 항공사 시스템 점검 시간대 접근
- **해결법**:
  1. ResyPro [시스템 상태] 탭에서 GDS 연결 상태 확인
  2. 연결 정상인 경우 → 해당 항공사 GDS 직접 접속(Amadeus/Sabre) 후 수동 조회
  3. 15분 후 재시도, 지속 오류 시 IT 지원팀(내선 4001) 연락

### 오류 2: 환불 금액이 실제보다 낮게 산출됨

- **증상**: ResyPro 자동 계산 금액이 고객 결제 금액보다 현저히 낮음
- **원인**: 유류할증료·세금 변동분이 시스템에 미반영되었거나, 프로모션 운임의 복합 페널티 규정이 잘못 적용된 경우
- **해결법**:
  1. ResyPro [운임 명세] 탭에서 원 발권 운임 구성 확인
  2. 항공사 공식 환불 규정서(ResyPro [항공사 규정 DB])와 수동 대조
  3. 차이 발생 시 [수동 입력] 모드로 전환 후 팀장 승인 득하여 처리

### 오류 3: 변경 처리 후 구 티켓이 시스템에 남아 있음

- **증상**: Reissue 완료 후 ResyPro에 구 티켓번호와 신 티켓번호가 동시에 활성 상태로 표시
- **원인**: 변경 처리 중 시스템 응답 지연으로 구 티켓 자동 무효화 단계가 누락됨
- **해결법**:
  1. ResyPro [티켓 이력] 탭에서 구 티켓 상태를 `VOID` 또는 `RFND`로 수동 변경
  2. 항공사 시스템에서도 구 티켓 무효 처리 여부 GDS를 통해 확인
  3. 처리 완료 후 [이력 메모] 란에 상황 요약 입력 필수

## 10. FAQ

**Q1. 고객이 출발 당일 공항에서 환불을 요청하면 어떻게 하나요?**
A. 공항 카운터에서 항공사 직접 처리를 안내하되, 반드시 사전에 예약·발권팀(내선 3100)으로 연락하여 PNR 분리 방지 조치를 취한다.

**Q2. 환불 처리 후 고객이 환불금 미수령을 주장하는 경우 어떻게 대응하나요?**
A. ResyPro [환불 이력]에서 RF번호와 처리일을 확인한 후, 해당 정보를 고객에게 제공하고 카드사 또는 은행 측 조회를 안내한다. 처리 후 15 영업일이 지나도 미수령이면 경리팀 협조 요청.

**Q3. 단체 예약 중 일부 인원만 환불 요청하는 경우 처리 방법은?**
A. ResyPro [그룹 PNR] → [부분 분리] 기능을 사용하여 해당 인원만 별도 PNR로 분리 후 환불 처리한다. 분리 전 반드시 팀장 승인을 받아야 하며, 나머지 단체 예약에 영향이 없는지 좌석 재확인을 실시한다.
