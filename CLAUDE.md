# CLAUDE.md — 롯데관광 AI 실무 강의 프로젝트

이 파일은 클로드 코드가 이 폴더에서 작업할 때 자동으로 읽는 마스터 컨텍스트입니다.
새 세션이 시작될 때마다 이 파일이 먼저 로드되어 전체 맥락을 파악합니다.

---

## 프로젝트 개요

- **프로젝트**: 롯데관광개발 임직원 대상 생성형 AI 실무 강의 자료 제작
- **강의 형식**: 50분 × 4차시 (4시간, 하루 연속 진행)
- **수강 대상**: 임직원 107명 (사전 설문 기준)
- **주력 도구**: Gemini (gemini.google.com) 무료 버전
- **보조 도구**: Claude, Perplexity (숙련자 보너스용)
- **계약 상태**: 체결 완료, 강의 일정 6월
- **강사**: 이대형

## 강의 핵심 메시지 (Narrative Spine)

> **"내 콘텐츠·내 문서·외부 정보·내 지식을 AI로 만들고 정리하는 4단계를 거쳐,**
> **4차시 끝에 본인 업무에 바로 쓸 결과물 4개를 손에 들고 나가는 강의"**

## 통합의 4가지 축 (차시 대응)

| 축 | 차시 | 핵심 도구 |
|---|---|---|
| 내 콘텐츠 통합 | 1차시 | Gemini + ROADS + Tona |
| 내 문서·외부 정보 통합 | 2차시 | @mention + Deep Research + 메일 마스터 |
| 내 지식 통합 | 3차시 | NotebookLM |
| 데이터·종합 응용 | 4차시 | Gemini 데이터 분석 + 종합 |

## 작업 시 반드시 지켜야 할 원칙

### 1. 톤·표현 원칙
- "오늘" 표현 금지 — "이번 차시" 또는 "여기까지"로 통일 (하루에 4차시 연속 진행이라 부자연스러움)
- 강의 톤은 **실무 중심·구체 예시·솔직한 한계 명시**
- 추상적·뻔한 표현 금지 (예: "환상적인", "꿈에 그리던" 같은 카피 자동 생성 결과는 Before 사례로만)

### 2. 도구 활용 원칙
- ChatGPT는 **호명만** 하고 본 시연 사용 안 함 (별도 심화 과정 안내)
- Gemini가 주력 도구 — 통합의 4가지 축 중 3개가 Google 생태계 연동
- Claude·Perplexity는 보너스 옵션 위치만
- **무료 도구만** 사용 (회사 라이선스 도입 결정 무관)

### 3. 데이터·보안 원칙
- **실데이터 절대 입력 금지** — 모든 실습은 더미 데이터
- 보안 가이드라인 **3분산** 적용:
  - 사전 안내 메일
  - 1차시 시작 1~2분 (삼성전자 ChatGPT 유출 사례)
  - 매 차시 시연 직전 10초 환기

### 4. 직군 균형 원칙
- 시연은 일본 패키지 기획자 시나리오 1개로 가더라도
- **실습은 7개 직군 모두 자기 시나리오로 진행**
- "노선 위주 운영" 우려를 차단하는 게 설계의 핵심

### 5. 마무리 활동 원칙
- 1·2·3차시: 가벼운 체크인 (1분) — "이번 차시 인상 깊은 것 / 써먹을 만한 것 / 부서에 필요한 것"
- 4차시: 종합 자기선언 + 부서별 1명 30초 발표

## 현재 작업 상태

### 완료된 것
- 사전 설문 분석 (107명)
- 4차시 구조 확정
- 1차시 블록 구성 확정 (A 오프닝 8분 / B 트렌드 7분 / C 프롬프트 8분 / D 시연 10분 / E 실습 12분 / F 마무리 5분)
- 롯데관광 회신 분석 (5/22) → 7가지 실제 업무 영역 확정
- 책 22개 항목 1차 분류
- 책 핵심 프롬프트 3종 확보 (프롬프트 마스터 / Tona / 메일 마스터)

### 미완성 (이 폴더에서 작업할 것)
- 7개 직군별 시나리오 카드 (28개 = 7직군 × 4차시)
- 더미 시나리오 4건 (롯데관광 검수 요청용)
- 강의계획서 v2 docx
- 2~4차시 상세 개요
- 1차시 D 블록 시연 시나리오 최종본 (Tona 기반으로 교체됨)
- 사전 안내 메일 초안

## 첫 작업 추천 순서

클로드 코드에서 이 폴더를 열면 다음 순서로 작업하는 것을 추천합니다.

1. **1차시 D 블록 Tona 기반 Before/After 시연 시나리오** 완성
   → 가장 임팩트 큰 부분, 강의 성패가 여기서 갈림

2. **더미 시나리오 4건** (신문광고·기획전·기안문·엑셀PPT) 작성
   → 롯데관광 검수 요청용, 외부 협조 필요한 자료

3. **7개 직군별 시나리오 카드 28개** 작성
   → 1~2번이 끝난 뒤 같은 패턴으로 확장

4. **2~4차시 상세 개요** 작성
   → 1차시 패턴이 확정된 다음에

5. **강의계획서 v2 docx** 통합
   → 모든 자료가 확정된 마지막 단계

## 강사 작업 스타일 (참고)

- 개요·마크다운 먼저 → 검토 → docx 최종본 순서를 선호
- 한 번에 큰 결과물 받기보다 단계별 확인하며 진행
- 추상적 항목 싫어함 — 모든 항목에 구체 예시 필수
- 단순화·압축 선호 — 복잡한 follow-up 시스템보다 간단한 체크인 선택
- 강사가 직접 시연 가능한 수준만 다룸 — 책 권위에 기대지 않음
